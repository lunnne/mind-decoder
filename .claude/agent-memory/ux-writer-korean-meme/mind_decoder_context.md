**Product**: 마음 해독기 (Mind Decoder) — Korean viral web tool where one user writes a long inner message, the system converts core words into consonant-only hints, and another user tries to decode it.

**Tone**: Silly, meme-like, absurd, cute, very Korean internet vibes (디시인사이드 energy, ㅋㅋ/ㄷㄷ culture)

**Target Users**: Korean users first, couples, friends, people who like playful guessing games

**Copy Guidelines**:
- Ultra-short, punchy (1-2 lines max per message)
- Screenshot-friendly, meme-like
- Natural Korean internet slang (ㅋㅋ, ㄷㄷ, ㅠ, ~야, etc.)
- Mix absurd humor with clear emotional tone
- Each result tier should have distinctly different emotional energy
- 1 emoji per message

**Result Screen Tiers** (score-based feedback):
- 100점: Shock, awe, exaggeration, "mind reading confirmed" vibe
- 80점+: Subtle praise mixed with suspicious teasing ("obsessed?" energy)
- 60점+: "Eh, okay, try harder next time" — wistful, mediocre
- 40점+: Disappointment, absurdity ("you really guessed this?"), half-joking
- 20점+: Sarcasm, suspicion, "did you just randomly type?" vibes
- 20점 미만: Devastation, friendship-ending territory, existential dread
